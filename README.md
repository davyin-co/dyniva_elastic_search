# 介绍
1. 默认支持elastic search IK中文分词。（可以配置分词模式，ik_smart, ik_max_word)。
2. 默认引入search_api_autocomplete, search_api_stats模块，可以实现高级搜索功能。
3. 自带elastic search 配置，可以通过直接修改elastic search 服务器配置来实现搜索页面。
